# Dependencias

Antes de ejecutar el programa, correr el comando `npm install` para instalar las dependencias necesarias.

# Ejecución

- Backend: Ejecutar `node index.js`.
- Frontend: Desde la carpeta front-end, ejecutar `npm run dev`.
